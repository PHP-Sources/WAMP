# WAMP

<img src="https://github.com/PHP-Sources/.github/raw/main/logos/wamp.webp" width="48" height="48" alt="WAMP">

**WAMP 3.3.0 (x32)** – La solution tout-en-un la plus populaire pour créer un serveur web local sous Windows (Apache + MySQL/MariaDB + PHP).

### Points forts
- Installation ultra-simple d’un environnement de développement complet
- Apache, PHP et MySQL/MariaDB préconfigurés
- Changement facile de version PHP, Apache et MySQL
- Interface de gestion (WampManager) très pratique
- Support de nombreuses extensions PHP
- Idéal pour le développement local de sites et applications web
